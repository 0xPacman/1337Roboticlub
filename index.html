About us

Robotic club is an extracurricular way for us to explore robotics in a more in-depth way

We are a team-oriented competitive club in which students work together to build robots and program them to compete in events against teams from other schools.

Blink code for Arduino

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);   // turn the LED on (HIGH is the voltage level)
  delay(1000);                       // wait for a second
  digitalWrite(LED_BUILTIN, LOW);    // turn the LED off by making the voltage LOW
  delay(1000);                       // wait for a second
}
